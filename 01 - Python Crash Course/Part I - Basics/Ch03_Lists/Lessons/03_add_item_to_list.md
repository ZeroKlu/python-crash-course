## Adding Item(s) to a List

You can add a new item to the end of a list using the `append()` function.

```python
motorcycles = ["honda", "yamaha", "suzuki"]
print(motorcycles)
motorcycles.append("triumph") # Add value to the list
print(motorcycles)
```

Output:

```
['honda', 'yamaha', 'suzuki']
['honda', 'yamaha', 'suzuki', 'triumph']
```
